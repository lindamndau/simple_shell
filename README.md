# Simple Shell Project

Author: Linda Mndau

## Overview

This is a simple shell project created to handle basic commands including `cd`, `exit`, `#`, and variable handling for `$` and `$$`.

## Features

### Supported Commands

- **cd:** Change directory.
- **exit:** Terminate the shell.
- **#:** Comment - allows the user to comment within scripts or commands.
- **$ and $$ Variables:** Handles variable substitution where `$` represents a single variable and `$$` represents the process ID of the shell.

## Usage

To use this shell:

1. Clone the repository.
2. Compile the shell program.
3. Run the compiled shell.

### Commands

- **cd [directory]:** Change the current working directory.
- **exit:** Terminate the shell.
- **#:** Add comments within scripts or commands.
- **Variable handling:** Utilize `$` for single variable substitution and `$$` for the process ID of the shell.

$ cd /path/to/directory
$ echo "Hello, World!"
$ # This is a comment
